This executable file is used to reproduce the experimental results from the paper.
IADGC_UpdateSequence is the version that changes the order in which edges are inserted and deleted.
IADGC_Parallel is a parallel version of IADGC.

How to Run
Please use the following command to run the executable file:
./IADGC -f "data.txt" -s 1
(Note: -s can be followed by a different random seed)

Output Results
The results will be saved in the output1.txt file.
